# PetFinder.my - Pawpularity Contest

This project implements Python notebooks and scripts that participate to the **PetFinder.my - Pawpularity Contest** competition on **Kaggle**. 

More information is available at: [https://www.kaggle.com/c/petfinder-pawpularity-score]

It contains the following files:
* a **Library** petfinder_pawpularity_lib.py
* a **Train Notebook** petfinder_pawpularity_train.ipynb
* a **Predict Notebook** petfinder_pawpularity_predict.ipynb
* a **Submit Notebook** petfinder_pawpularity_submit.ipynb
